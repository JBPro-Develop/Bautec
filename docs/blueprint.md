# **App Name**: BAU-TEC Farm Manager

## Core Features:

- Dashboard Overview: Displays a summary of pens, animal count, current recipes, last feeding date, and pen status.
- Daily Feeding Management: Allows users to select a pen, view/adjust the feeding recipe, and record feeding details.
- Health and Treatment Records: Manages health records for each pen, adding treatments, dosages, and setting treatment reminders.
- Pen/Group Closure: Handles the closing process for a pen, calculating metrics like total gain, feed conversion ratio, and profit/loss, then exporting data as PDF or CSV.
- New Pen/Group Creation: Form to create a new pen with details such as name, number of heads, arrival date, expected ship date, and initial average weight, along with animal tags and recipe selection.
- AI-Powered Feeding Insights Tool: Generates recommendations for adjustments to feeding recipes based on current conditions, past performance, and predictive models for optimal growth, notifying users when to change a recipe to maintain performance within a selected pen or set of pens. Uses reasoning about farm conditions, feed price and ingredient availability.

## Style Guidelines:

- Primary color: Soft beige (#F5F5DC), evoking a natural, earthy tone.
- Background color: Very light gray (#F0F0F0), providing a neutral backdrop.
- Accent color: Muted olive green (#808000), complementing the primary with an analogous color that conveys health and growth.
- Body and headline font: 'PT Sans', a versatile humanist sans-serif for readability and a modern touch.
- Use simple, clear icons related to farm activities, health, and reporting.
- Dashboard should use a grid layout, dividing information logically and intuitively.
- Subtle transitions when updating records and displaying reports, to give a sense of lively system.
